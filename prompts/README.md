# Promptit: huoltokirja

Tassa kansiossa on valmiit promptit AI:lle, jotta projektia voi vieda eteenpain hallitusti vaihe vaiheelta.

## Suositeltu jarjestys

1. `01-projekti-analyysi-suunnitelma.md`
2. `02-android-ui-viimeistely.md`
3. `03-nosql-migraatio.md`
4. `04-ios-parity.md`
5. `05-laatu-release-readiness.md`

Jos haluat ajaa koko prosessin yhdella promptilla, kayta:

6. `06-superprompt.md`

## Kayttotapa

1. Avaa haluttu tiedosto ja kopioi koko sisalto AI-keskusteluun.
2. Aja vaiheet yksi kerrallaan (suositus), jotta muutokset pysyvat pienina ja tarkastettavina.
3. Hyvaksy checkpointit vaiheittain ennen seuraavaa vaihetta.
4. Pyyda jokaisen vaiheen lopussa:
   - muutetut tiedostot
   - testausohjeet
   - avoimet riskit

## Vinkki

Turvallisin eteneminen on ajaa ensin `01-05` jarjestyksessa. `06-superprompt.md` sopii tilanteeseen, jossa haluat AI:n johtavan koko toteutusta checkpoint-mallilla.
